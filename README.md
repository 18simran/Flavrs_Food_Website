## Overview

Flavrs Food is crafted with a passion for food and technology. The goal of the application is to provide an integrated experience for users who want to:
- Discover new recipes using an intuitive search functionality.
- Understand the nutritional value of each recipe.
- Watch high-quality video tutorials to enhance their cooking skills.

---

## Features

- **Recipe Search**: Quickly search for recipes by name, ingredients, or cuisine.
- **Nutrient Details**: View detailed nutritional information for each recipe, including calories, proteins, fats, and more.
- **Video Tutorials**: Watch step-by-step video guides on how to prepare various dishes.
- **Responsive Design**: Enjoy a seamless experience on both desktop and mobile devices.
- **User-Friendly Interface**: Simple and attractive layout with smooth navigation.
- **Interactive UI Elements**: Engaging animations and effects powered by JavaScript.

---

## Tech Stack

- **HTML5**: For structuring the content.
- **CSS3**: For styling and responsive design.
- **JavaScript (ES6+)**: For interactive and dynamic functionality.

## Deployment

Flavrs Food is hosted on Netlify, ensuring fast load times and reliable uptime. The live version of the site can be accessed at:

[https://foodflavrs.netlify.app/](https://foodflavrs.netlify.app/)

Netlify handles continuous deployment, so every push to the repository automatically updates the live site.
